# PROJECT_OVERVIEW_V1.0

## 项目名称

韩国星际争霸团战数据自动采集与统计平台

英文名称：

StarCraft Team League Data Platform

简称：

STDP（StarCraft Team Data Platform）

------

# 1. 项目背景

韩国星际争霸（StarCraft: Remastered）职业团战赛事长期活跃。

目前最主要的数据来源之一为：

ELOBOARD职业团战专区。

```text
https://eloboard.com/men/bbs/board.php?bo_table=pro_league
```

该网站每日发布：

- 团战结果
- 对阵信息
- 选手数据
- 奖金数据
- 地图信息

但存在以下问题：

## 数据分散

历史数据以网页形式保存。

无法进行长期统计分析。

------

## 韩文阅读门槛高

主要内容为韩文。

中文用户阅读和使用成本较高。

------

## 缺乏统一数据库

无法快速查询：

- 选手历史战绩
- 地图胜率
- 奖金排名
- 队伍表现

------

## 无法形成长期数据资产

历史比赛持续增加。

但无法沉淀为结构化数据。

------

# 2. 项目目标

建设一个长期稳定运行的数据平台。

实现：

```text
自动采集
    ↓
自动翻译
    ↓
结构化存储
    ↓
统计分析
    ↓
网页展示
```

最终形成：

韩国星际争霸职业团战数据库。

------

# 3. 项目定位

本项目定位为：

## 电竞数据平台

而不是：

```text
新闻网站

内容平台

自媒体工具

公众号工具
```

核心价值：

数据积累与数据分析。

------

# 4. 核心功能

## 自动采集

每日自动获取：

ELOBOARD最新团战数据。

------

## 自动翻译

依据：

translate_rules.md

完成：

- 选手翻译
- 地图翻译
- 队伍翻译

------

## 数据管理

自动建立：

- 选手库
- 地图库
- 队伍库
- 赛季库

------

## 数据统计

支持：

- 胜负统计
- 连胜统计
- 奖金统计
- 地图统计
- 队伍统计

------

## 数据展示

提供：

- 首页
- 选手页
- 队伍页
- 地图页
- 排行榜页
